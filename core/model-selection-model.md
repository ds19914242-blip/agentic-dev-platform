# Model Selection Model

## Definition

Model Selection determines which AI model is assigned to an Agent execution.

## Principle

Agents define capabilities.

Models provide execution.

Agents and models are not the same thing.

## Examples

Architect Agent
→ Claude

Research Agent
→ GPT

QA Agent
→ GPT

Security Agent
→ Claude

## Selection Factors

Model selection may consider:

- task complexity
- context size
- reasoning requirements
- cost
- latency
- reliability
- risk level

## Responsibility

The Orchestrator selects models.

Agents do not select their own models.

## Rules

Model choice must be recorded.

Model changes must be auditable.

High-risk decisions may require stronger models.

The system should support multiple model providers.
