# Agent Types

## Planning Agents

Responsible for understanding requests and creating execution plans.

### Planner Agent

Transforms requests into execution graphs.

### Product Agent

Analyzes business requirements, user impact and product scope.

### Architect Agent

Analyzes architecture, dependencies, risks and technical approach.

---

## Engineering Agents

Responsible for implementation and code changes.

### Backend Engineer Agent

Implements backend functionality.

### Frontend Engineer Agent

Implements frontend functionality.

### Infrastructure Engineer Agent

Implements infrastructure and deployment changes.

### Database Engineer Agent

Implements schema and database changes.

---

## Validation Agents

Responsible for review, testing and verification.

### Code Reviewer Agent

Reviews code quality and maintainability.

### QA Agent

Validates functionality against requirements.

### Security Agent

Reviews security implications and vulnerabilities.

### Performance Agent

Reviews performance and scalability impact.

---

## Operations Agents

Responsible for deployment, monitoring and infrastructure operations.

### Release Agent

Prepares releases and release notes.

### Deployment Agent

Manages deployment workflows.

### Monitoring Agent

Analyzes logs, metrics and incidents.

---

## Knowledge Agents

Responsible for memory, documentation and knowledge management.

### Documentation Agent

Maintains technical and product documentation.

### Memory Agent

Maintains system memory and knowledge records.

### Research Agent

Performs research and gathers information.
