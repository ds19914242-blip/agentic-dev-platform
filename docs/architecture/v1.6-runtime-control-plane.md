# v1.6 Runtime Control Plane

## Added

P1 Acceptance / Playwright end-to-end support:
- runtime acceptance can run with --epic-dir
- scripts/runtime_full_acceptance.py can execute runtime + optional epic acceptance

P2 Recovery Loop:
- executor emits recovery events
- validation failure can trigger implementation + validation retry with --recovery

P3 Runtime Live Monitoring:
- runtime-events.jsonl
- runtime-timeline.md
- agentic.py runtime-monitor <run_dir>

P4 Release Verification:
- runtime full acceptance script can call release-check when --execute-release and --epic-dir are provided

## Important

Real acceptance requires an epic directory with acceptance-scenarios.md or acceptance-command.txt.

Real release verification requires a task path inside an epic and should only be run after deployment.
