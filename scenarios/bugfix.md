# Scenario: Bug Fix

## Example Request

Fix a production bug.

## Questions

- How is the bug reproduced?
- Which repositories are affected?
- Which environments are affected?
- Which agents investigate the issue?
- Who validates the fix?
- What happens if the fix fails?
- Is rollback required?
- Is approval required?

## Expected Outcome

The bug is fixed, validated, documented and recorded in memory.
